GIT

  • Enter your GIT notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A stages and commits all files in the working branch
  • git commit -m "ex.added code to HTML file" will associate a message with my commit
  • git pull origin main checks if your code is up to date
  • git push origin "ex.feature/add-html" pushes it to git hub/main branch
